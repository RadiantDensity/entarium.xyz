# Entarium.xyz 🌐  
The radiant frontend portal for creators to upload, manage, and distribute essence across the Entarium network.

## Tech Stack
- Next.js / Vite / Tailwind CSS
- IPFS-integrated file handling
- Wallet-based auth (planned)

## Commands
```bash
npm install
npm run dev
```

## Structure
- `/pages`: Main routes
- `/components`: Reusable UI
- `/lib`: Web3 + API utils
